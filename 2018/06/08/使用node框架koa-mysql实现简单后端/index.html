使用node框架koa+mysql实现简单后端开发

基于node的koa框架，创建服务，连接msql服务器实现较为简单的后端开发，如增，改，删，查；
学习的时候用的资料是https://blog.csdn.net/wclimb/article/details/77890793，资料上面是用ejs渲染的网页，个人目标是能不能用koa+mysql构建后端，实现简单的登录，注册，删除，添加等基本功能；然后是用vue + webpack搭建前端，用koa+mysql搭建后端，加油！！
